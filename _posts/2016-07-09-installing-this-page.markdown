## Configuration

홈페이지 생성을 원하는 디렉토리로 이동하여 다음과 같이 입력함.

```
$ jekyll new . --force
```

## Testing
```
$ jekyll serve
```

브라우저로 `localhost:4000`로 접속하면 페이지를 확인할 수 있다.
